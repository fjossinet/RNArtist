The help mode in RNArtist

With the help mode activated, a window will be displayed after a mouse click on some graphical elements in RNArtist. The help window will explain shortly the goal of the graphical element. At the bottom, you will find a book icon that will redirect you to the documentation where you will find more details.

You can desactivate this help mode by clicking one more time on its button. You can also browse the full documentation by clicking "Table of contents" in the navigation bar at the top.
