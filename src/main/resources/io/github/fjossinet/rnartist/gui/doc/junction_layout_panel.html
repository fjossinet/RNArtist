The junction layout panel

This panel allows you to change the layout (size and helices orientation) for junctions.

Navigation

This allows to scroll over the different junctions in the current selection and to highlight a different one. When a junction is highlighted, the knob below displays its current layout.

For more details about the highlighting of an RNA object, check this documentation page

Layout Knob

  • each circle is a direction for an helix according to a compass. If the circle is orange, this is the current orientation for one of the helices in this junction. Click on an orange circle to unselect it and choose another one to define a new direction.
  • left/right arrows move helices in anti/clockwise direction
  • up/down arrows increase/decrease the size of the junction
