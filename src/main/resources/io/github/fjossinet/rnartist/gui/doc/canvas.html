The canvas 2D

The canvas 2D can be divided into:

  • its drawing surface
  • its toolbars

Quickstart

  • use the right mouse button to translate the full 2D
  • use the buttons panel on the left to zoom in/out and recenter on the full 2D
  • use the left mouse button to select a 2D object
    • click on the center of an element to select it (residue, junctions, helices, interactions, phosphodiester bonds,...)
    • successive clicks on a residue select the next parent element:
      • residue -> secondary interaction -> helix -> former structural domain in the branch
      • residue -> junction -> former structural domain in the branch
      • residue -> single-strand
  • once a selection done, you can modify its theme with the right theme panel
  • if you select junctions, you can modify their layout with the layout panel

The drawing surface

Select parts of an RNA 2D

You have two different ways to make a selection with RNArtist:

  • using these buttons in the theme panel on the right
  • by clicking on the drawing surface

You can select any rna objects with a left-click inside its bounds to add it to the current selection. You can do successive left-click on the same residue to enlarge the selection (for example residue -> secondary interaction -> helix -> next structural domain along the branch).

To clear the selection, you can either double left-click on the blank surface or use this button in the theme panel on the right

The highlight of an RNA object in the current selection

When several rna objects are selected, choosing theme parameters with the theme panel on the right will target all of them at once. But sometimes, you could need to:

  • target a single object while keeping the selection as it is
  • apply different theme parameters to each object in the selection.

This is why RNArtist allows you to highlight an rna object in the selection in order to make it the sole target of your choices. When several rna objects are selected without any one highlighted, all the shapes around them display the same color. When you use these buttons, you're highlighting one selected object after another one. The shape of the highlighted rna object gets its own color, while the other shapes are drawn with a lighter gray.

The toolbars

The current drawing bar

  • use this button to go to the folder where the current drawing has been saved
  • use this button to delete the current drawing

The navigation bar

  • use this button to zoom in
  • use this button to zoom out
  • use this button to fit the full 2D into the current view
  • use this button highlights and centers the view on the former selected element
  • use this button highlights and centers the view on the next selected element

The undo/redo bar for theme

The undo/redo bar for layout

The save/export bar

  • use this button to save your 2D. The current layout and theme will become the new start in their corresponding undo/redo history. This button is enabled if you have a 2D displayed in the canvas and a folder from your database selected. The 2D will be saved in this folder.
  • use this button to apply the theme of the current 2D to all the other 2Ds displayed in the database explorer. Like the former button, this button is enabled if you have a 2D displayed in the canvas and a folder from your database selected.
  • use this button to export the current 2D in an SVG file. There is no need to have the 2D saved to be able to export it. So you can for example generate a 2D with the bracket notation panel, change its layout and theme and then export it in an SVG file.

The lateral panels bar

With these buttons you can display/hide the left, right and bottom panels to have more space for your 2D.
