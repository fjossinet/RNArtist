The theme panel for selections

With this panel you can select RNA objects in the 2D drawing and modify their features (color, line width, details level). If you prefer to act on the full 2D, you need to use the theme panel on the left side

In fact, RNArtist provides two different ways to make a selection:

  • using the selection buttons in this panel
  • using mouse clicks on the drawing surface of the canvas 2D

Selection

  • this button makes a selection according to the type defined here. The search is restricted to the location defined by the current selection in the 2D canvas (if any) or defined in the location subpanel below.
  • this button makes a selection according to the type defined in this list and adds it to the current selection (if any). The search is done on the full 2D.
  • this button reverses the current selection. It selects the RNA objects not present in the current selection, and corresponding to the type defined in this list. The search is done on the full 2D.
  • this button clears the current selection.
  • this subpanel allows to restrict the selection to a user-defined location. It has the priority over any selection made directly in the 2D canvas (if any). The location is computed from the current list of intervals.
    • this button adds a new interval to the list
    • this button recovers the location from the current selection in the 2D canvas and add it to the list of intervals
    • this button clear the list of intervals

Details level

  • with this button, the selected RNA objects become lowly rendered. Each additional click does the same for the next parent of the selected elements.
  • with this button, the selected RNA objects become highly rendered. Each additional click does the same for the next children of the selected elements.

For more details about the concept of details level, check this documentation page

Color & Lines

  • by clicking on the color wheel, you define a new color for all the selected objects. This adds the color to the last colors panel below the wheel, along with colors with different levels of saturation and brightness.
  • click on this button before to choose the color to apply it to the residue characters/letters
  • click on this button to recover the color of the RNA object highlighted
  • by clicking on the knob surrounding the color wheel, you rotate it and define the new line width for all the selected objects
  • by clicking on this button on the knob, you apply the line width corresponding to its current position.
