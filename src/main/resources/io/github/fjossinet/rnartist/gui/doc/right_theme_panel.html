The theme panel for selections

This panel targets the rna objects selected in the canvas. To target the full 2D, you need to use this panel

You have two different ways to make a selection with RNArtist:

  • using the selection buttons in this panel
  • by clicking on the drawing surface of the canvas 2D

Selection

  • this button makes a selection according to the type defined here. If a prior selection exists, the search is be restricted to the location defined by the current selection.
  • this button reverses the current selection. It selects the rna objects not present in the current selection, and corresponding to the type defined here.
  • this button makes a selection according to the type defined here and adds it to the current selection (if any)
  • this button clears the current selection.

Details level

  • with this button, the selected rna objects become lowly rendered. Each additional click does the same for the next parent of the selected elements.
  • with this button, the selected rna objects become highly rendered Each additional click wdoes the same for the next children of the selected elements.

For more details about the concept of details level, check this documentation page

Color & Lines

  • by clicking on the color wheel, you define a new color for all the selected objects. This adds the color to the last colors panel below the wheel, along with colors with different levels of saturation and brightness.
  • click on this button before to choose the color to apply it to the residue characters/letters
  • click on this button before to pick the color of the rna object highlighted
  • by clicking on the knob surrounding the color wheel, you rotate it and define the new line width for all the selected objects
  • by clicking on this button on the knob, you apply the line width corresponding to its current position.
