The concept of RNA 2D in RNArtist

In RNArtist, a secondary structure is a hierarchy of 2D objects (helices, junctions, residues,...). Each 2D object be associated to its own theme (colors, line width, details level) and a junction also have a specific layout.

The theme

Color and Lines

The color and the width/thickness of the line can be defined.

Details level

The details level allows to define if a 2D object is displayed with a high or low level of details:

  • low level: the 2D object doesn't allow its children to be displayed (whatever their own details level). The 2D object renders itself with a default representation (for exemple a circle for a junction)
  • high level: in general, the 2D object doesn't render itself and allows its children to be displayed (whatever their own details level).

The layout

The layout of a junction is defined by:

  • its radius
  • the position of its connected helices according to a compass
