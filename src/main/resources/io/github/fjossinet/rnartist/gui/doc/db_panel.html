The database explorer panel

Quickstart

  • choose a folder as an RNArtist database with this button (more about the concept of RNArtist database here)
  • once the database loaded, select a folder in the tree to compute and load the 2Ds. If you don't have any folder in your database, select the root folder, and click on this button to create a first one
  • left click on a thumbnail to load its 2D in the canvas
  • if you don't have any structural files in the folder selected, you can:
    • create, edit and and save new 2D structures with the bracket notation panel
    • download and save data in this folder outside of RNArtist, then reselect this folder from within RNArtist
    • load an RNACentral entry in your browser, drag and drop the web address from your browser into the subfolder in RNArtist. The data will be automatically downloaded, added to the subfolder. The 2D drawing will be computed and displayed as a thumbnail.
  • following the loading of your database, if you add new folders with structural data outside of RNArtist, then you can index and display them in RNArtist using this button
