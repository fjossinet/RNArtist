The database explorer panel

RNArtist allows you to gather, visualize and manage all your RNA 2Ds in a collection (named an RNArtist database). You have different ways to start an RNArtist database:

  • if you already have a folder with subfolders containing structural data (Vienna/BPSeq/CT files), use this button to choose this folder as a new database
  • create a new folder on your computer and choose it with this button to start a new empty database

Once loaded, you have different ways to fill your database:

  • create an RNA 2D with the bracket notation panel and save it in your database
  • move data into the subfolders of your database outside of RNArtist, then use this button to sync your database
  • load an RNACentral entry in your browser, drag and drop the web address from your browser into a folder in your database. The data will be automatically downloaded, plotted and added to your database.

Description of the database panel
