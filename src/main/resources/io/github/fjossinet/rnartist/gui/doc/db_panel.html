The database explorer panel

Quickstart

  • choose a folder as an RNArtist database with this button (more about the concept of RNArtist database here)
  • once the database loaded, select a subfolder in the tree and use this button to compute and load the 2Ds. If you don't have any subfolder in your database, select the root folder, and click on this button
  • left click on a thumbnail to load its 2D in the canvas
  • if you don't have any structural files in the subfolder selected, you can:
    • download and add data outside of RNArtist, select this folder in RNArtist and load it with this button. RNArtist will compute the drawings for the 2Ds found in the folder and preview them with thumbnails
    • load an RNACentral entry in your browser, drag and drop the web address from your browser into the subfolder in RNArtist. The data will be automatically downloaded, added to the subfolder. The 2D drawing will be computed and displayed as a thumbnail.
  • following the loading of your database, if you add new subfolders with structural data outside of RNArtist, you can index and display them in RNArtist using this button
