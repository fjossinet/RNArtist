The theme panel for the full 2D

This panel allows you to chosoe a theme that will target all the RNA objects in the 2D. To target selected RNA objects, you need to use this panel

Details level

Each button defines the details level for the full 2D (from 1 to 5):

  • level 1: all the objects in the current 2D are lowly rendered
  • level 2: helices, secondary-interactions, junctions, single-strands and phosphodiester bonds are highly rendered
  • level 3: residue shapes are highly rendered
  • level 4: residues letters are highly rendered
  • level 5: interaction symbols are highly rendered

For more details about the concept of details level, check this documentation page

Color & Lines

  • the scheme menu applies different color schemes to the full 2D
  • by clicking on the color wheel, you define a new color for all the RNA objects in your 2D. This also adds this color to the last colors panel below the wheel, along with colors with different levels of saturation and brightness.
  • click on this button before to choose the color to apply it to the residue characters/letters
  • click on this button before to choose the color to apply it to the rna objects that are not selected after a selection
  • by clicking on the knob surrounding the color wheel, you rotate it and define a new line width for for all the RNA object in your 2D
  • by clicking on this button on the knob, you apply the line width corresponding to its current position.
