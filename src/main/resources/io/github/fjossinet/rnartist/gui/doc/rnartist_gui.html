The graphical user interface

RNArtist is made with:

  • the upper part divided into:
    • a left panel containing:
      • a theme panel to change the theme for the full 2D
    • the canvas where the 2D structure is drawn
    • a right panel containing:
      • a theme panel to change the theme for the current selection in the 2D
      • a junction layout panel to change the layout for the junctions selected in the 2D
  • the lower part divided into:
    • a bracket notation panel to create a 2D structure from scratch using the dot-bracket notation
    • a database panel to load and manage RNA 2Ds stored in an RNArtist database
    • a charts panel to plot experimental data linked to the RNA 2D displayed in the canvas (to come)
    • a documentation panel
