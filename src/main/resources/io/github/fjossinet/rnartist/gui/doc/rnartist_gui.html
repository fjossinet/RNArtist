The graphical user interface

RNArtist is made with:

  • an upper panel divided into:
    • an upper-left panel containing:
      • a panel to change the theme for the full 2D
    • the canvas where the 2D is painted
    • an upper-right panel containing:
      • a panel to change the theme for the current selection in the 2D
      • a panel to change the layout for the junctions selected in the 2D
  • a lower panel containing:
    • a panel to load and manage RNA 2Ds stored in an RNArtist database
    • a panel to plot experimental data linked to the RNA 2D displayed in the canvas (to come)
    • a panel to get the documentation and to do some tutorials
