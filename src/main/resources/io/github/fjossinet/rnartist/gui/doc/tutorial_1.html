Tutorial 1

Introduction

In this tutorial, you will learn the basic use of RNArtist. You will create your first database and download your first RNA 2D. Then you will learn how to modify the theme and layout of this structure, how to save it and how to export it as an SVG file.

Don't click on the book icon, or you could leave this tutorial for another page in the documentation. Just click on the top-right icon to close the window and to pursue. You can also desactivate the help mode during the tutorial if you prefer. This tutorial is linked to the graphical interface of RNArtist. Some links will highlight the corresponding widget.

Part 1: your first RNArtist database and your first 2D

  1. create a new directory on your computer and use this button to choose it.
  2. select this directory in RNArtist and use this button to create a subfolder.
  3. RNArtist is about to download a first structure from RNACentral into your new subfolder. To do so, select the subfolder, come back to this tutorial and click here
  4. once done, go back to your database explorer. You should see something like that:
  5. Now click on this thumbnail to load the 2D into the canvas.

Part 2: manipulation and theming of the full 2D

If you need a fresh start to practice part 2, click here to compute the prerequisites.
  1. click on this button to change the details level of the full 2D. Try the four other ones.
  2. the differences between details level 3 and 4 are mainly the residue characters/letters. Zoom in with this button to have a better insight into these differences.
  3. to translate your 2D, keep the right mouse button pressed on the canvas and move your mouse
  4. to recenter your 2D on the canvas and to restore the initial zoom level, click on this button
  5. to choose a new color for all the rna objects in a 2D, you can:
    • choose a color scheme
    • choose a color by clicking on the color wheel on the left. If you want to color all the residue letters, you have to click on this button before to choose the color.
  6. to choose a new line width for all the rna objects in a 2D, click on the circular knob around the color wheel.
  7. you can also click the button on the circular knob to apply the corresponding line width.

Part 3: selection and theming of 2D parts

If you need a fresh start to practice part 3, click here to compute the prerequisites.

Now, we would like to color all the apical loops in green and the remaining 2D in blue

  1. click here to have the full 2D at details level 3 and everything in blue
  2. choose "Apical Loops" in the selection types
  3. click on this button to make your selection
  4. choose a green color by clicking on the color wheel on the right
  5. double click on the canvas to clear the selection or click on this button

Now, we would like to color all the other junctions in purple.

  1. select the apical loops as before
  2. to indicate to RNArtist that we want to select all the other junctions, choose "Junctions" in the selection types
  3. use this button to reverse the current selection. It selects the rna objects not present in the current selection, and corresponding to the type required.
  4. choose a purple color and unselect everything.

Now, we would like to color the secondary interactions A-U in orange.

  1. first, select the adenines as you did for apical loops (choose the type and click on select. Don't forget to start without any prior selection, ortherwise the adenines found will be restricted to this location)
  2. to add the uracils to the current selection, choose the type "Uracils" and then use this button. You should get this result:
  3. if we already have a selection, this button restricts its search to the location defined by this selection. Consequently, if you choose now the type "Secondary Interactions" and you click on this button, you will only get the interactions made between a selected A and a selected U.
  4. color the remaining selection in orange

Part 4: the undo/redo history for theme

If you need a fresh start to practice part 4, click here to compute the prerequisites.

To scroll through the steps of your theme choices:

  • use this button to go back to the first step
  • use this button to go back to the former step
  • use this button to go back to the next step
  • use this button to go back to the last step

When you save your work, RNArtist creates a new undo/history with the current theme as the first step.

This last button allows you to automatically scroll through the steps in your history. This is still an experimental feature that should lead to the ability for RNArtist to produce animations.

Part 5: modification of junction layouts

  1. this structure contains several 3-Way junctions. We will use the largest one. You have different ways to select it (see the videos below):
    • by clicking on its center in the canvas
    • by clicking on one of its residue, than be doing a second click on the same residue to select its parent
    • by asking RNArtist to select all the 3-Way junctions, and by highlighting the one we want with these buttons (if your selection is made with a single junction, it is automatically highlighted)

    2. (if you cannot play the videos below, please install the H.264 codec on your computer)

  2. go to the junction layout panel
  3. when a junction is "highlighted", its layout is available for modification in this panel
  4. to change its size, you can click on these arrows
  5. to change the orientation of the helices, you can:
    • either use these arrows to rotate all the helices clockwise or anticlockwise
    • or click on a orange circle and then on an empty one to change the orientation of the corresponding helix. If you have unselected several colored circles, the layout will be applied to the junction only when you will recover the initial number of colored circles.

Part 7: saving and export

  1. click on this button to save all the changes you made for the current 2D. This will update its preview in the database explorer. If you have undo some modifications, they will be deleted during the save. The current theme and ayouts will become the new first step in your undo/redo history.
  2. click on this button to export your 2D in an SVG file.

Part 8: more structures in your database

  1. click here to let RNArtist create several folders in your database and download several structures from RNACentral
  2. once done, get a look at your database explorer. When you select a folder in the tree, you compute and load all the 2Ds stored in this folder.
